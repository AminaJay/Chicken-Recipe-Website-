# Chicken-Recipe-Website-

## Project Description
This project is a dynamic, responsive Recipe Website built to fetch and display meal data from TheMealDB API. Focusing on a mobile-first approach, the site presents recipes in clean, adaptable cards featuring the meal's image, title, instructions summary, top 5 ingredients, and a direct link to a YouTube video for cooking instructions. The site demonstrates proficiency in consuming and manipulating JSON data and designing a visually appealing, cross-device interface.

***
## API Endpoint Used
The project utilizes the following endpoint from TheMealDB API to retrieve meal data of this project here:
[Live website](https://www.themealdb.com/api/json/v1/1/search.php?s=chicken)
(Note: The search term can be easily modified in script.js to retrieve other meal types, e.g., s=rice or s=beef.)

## Installation
This is a static website built with HTML CSS and JavaScript. To view it, simply open the index.html file in your web browser. For the best experience, view it using a browser's mobile device simulator.

***

## Usage
The Meal Finder website allows users to:
View a curated list of recipes (default is "Chicken").
See the meal's image and a brief summary of the instructions.
Review a list of the top 5 key ingredients and measurements.
Access video instructions directly via the "Watch Video" button.
Experience a seamless design on any device (mobile, tablet, or desktop).
